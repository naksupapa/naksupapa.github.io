# Claude가 만든 테스트 페이지

이 페이지는 Claude Code에 의해 자동 생성된 테스트 페이지입니다.

## 마크다운 기능 테스트

### 텍스트 스타일

- **굵은 글씨** 테스트
- *이탤릭* 테스트
- ~~취소선~~ 테스트
- `인라인 코드` 테스트

### 순서 있는 목록

1. 첫 번째 항목
2. 두 번째 항목
3. 세 번째 항목

### 체크리스트

- [x] 완료된 항목
- [ ] 미완료 항목
- [ ] 또 다른 미완료 항목
